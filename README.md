# LitHeaven

An Ecommerce Website for purchasing books online.

## Features

- User Authentication
- Google Authentication
- Product Management
- Cart Management
- Wishlist
- Wallet System
- Razorpay Payment
- Order Management
- Coupon Management
- Sales Report
- Referral System

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- Cloudinary
- Razorpay

## Installation

```bash
npm install
```

## Environment Variables

Create a .env file and add:

```env
PORT=
MONGO_URL=
SESSION_SECRET=

EMAIL=
EMAIL_PASS=

CLOUD_NAME=
API_KEY=
API_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_CALLBACK_URL=

RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=
```

## Run Project

```bash
npm start
```
